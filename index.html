About Clear Sans

Clear Sans is an open-source typeface designed by the Intel® Open Source Technology Center, and is released under the Apache 2.0 license. This is the webfont conversion available in TrueType, WOFF, EOT and SVG format, which ensures that you have the best format for the font to be displayed smoothly on any system.

Feel free to expand on this project by forking it on GitHub. Giving back is highly encouraged.
